FAQ

Is ALCIG free of use?
No, but if you register you will be able to use ALCIG for free a maximum of three times. If you are then interested to buy a license, you have to contact Quantis to get additional information.
What is the pricing?
Please contact us and we will be happy to send you additional information by e-mail or organize a phone discussion.
How can I learn more about the methodology that is used?
ALCIG is following the methodological guidelines of the World Food LCA Database. You can find more information on the project on the following website :
https://quantis-intl.com/tools/databases/wfldb-food/
The methodological guidelines are available using the following link:
https://quantis-intl.com/wp-content/uploads/2017/02/wfldb_methodologicalguidelines_v3.0.pdf
What background data are used for the calculation in ALCIG?
The tool uses ecoinvent data.
What unit of reference is used for the results?
All results are given for the production of one hectare of the crop you have selected.
Can ALCIG be used for other crops than those that are in the predefined list?
No, it is currently not possible to assess other crops than those that are listed in the data collection sheet. If you need to assess other crops, contact us and we will evaluate the feasibility to add them.
Will other output format than Simapro be available?
Currently, only Simapro format and ecoinvent background data are available. In case you would like to have other background data used or other output format available, feel free to contact us so that we can evaluate the feasibility and make sure to inform you when the tool will integrate them.
If I have comments, suggestions or questions, what should I do?
If you have comments, suggestions or questions, feel free to contact Quantis and we will make sure to take them into account and send you an answer. You can either use the on-line questionnaires or send us an e-mail : {{quantisEmail}}
I don't manage to import the generated CSV file into Simapro:
Displayed error: "Unknown CSV format version (7.0.0},,,,,,,,,,,)."
You may have opened the generated CSV file with Microsoft Excel. This software adds some unwanted characters that corrupt the generated file. Please redo the export from ALCIG and import it directly into Simapro, without opening it with an other software.
Displayed error: "One or more format options not found (CSV Format version, [...]). For these options default values will be used. Continue?"
You may have received this file by email from someone using Apple Mail. This software adds some unwanted characters that corrupt the generated file. Please ask the sender to send a ZIP archive of the CSV file.

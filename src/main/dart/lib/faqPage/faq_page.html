FAQ

What is meant by a beta version?
ALCIG is finalized and should work properly. It is however still in beta version as a full set of quality check has not been performed so far. ALCIG is therefore used at the own risk of the users and results should be carefully checked.
When is ALCIG going to be finalised?
The first version of ALCIG should be available early September. The beta version will be available for free until then.
Will there be documentation available?
Yes, a full methodological report will be available together with the tool. In addition, a forum will be implemented for users to shared knowhow, comments and information. The forum will be available only for the users of the tool.
Will ALCIG be free of use?
No, ALCIG will be accessible in exchange of a licence. The pricing will be communicated to people interested.
What background data are used?
The tool uses ecoinvent data.
Will other output format be available?
Currently, only Simapro format and ecoinvent background data are available. In case you would like to have other background data used or other output format available, feel free to contact us so that we can evaluate the feasibility and make sure to inform you when the tool will integrate them.
If I have comments and suggestions, what should I do?
If you have comments, suggestions or questions, feel free to contact Quantis and we will make sure to take them into account and send you an answer.
I don't manage to import the generated CSV file into Simapro. Displayed error: "Unknown CSV format version (7.0.0},,,,,,,,,,,)."
You may have opened the generated CSV file with Microsoft Excel. This software adds some unwanted characters that corrupt the generated file. Please redo the export from ALCIG and import it directly into Simapro, without opening it with an other software.

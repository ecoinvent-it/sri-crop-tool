FAQ

What is the aim of the “Ecoinvent LCI calculation tool for crop production”?
Aim of this tool is to facilitate the creation of Life Cycle Inventory data for agriculture. The tool creates a spold 2 document containing an agricultural inventory compliant with the ecoinvent guidelines. It maps the inputs and outputs with ecoinvent activities and calculates the field emissions based on the data filled in the template.
Can I calculate impact results with the tool?
No, you cannot calculate impact results with the tool, but you can create a dataset in a LCA software based on the spold document.
How can I enter my agricultural data?
Please enter your data in the excel template you can download from the webpage and follow the guidance of the corresponding tab.
Is there default data in the tool?
The tool does not provide automatic default data.
Where can I find the methodological background of the tool?
The methodological background are the data quality guidelines of ecoinvent as well as following document describing the models used in the tool.
What format will be delivered?
The tool only delivers an inventory in ecospold 2 format.
What unit of reference is used for the results?
The reference unit is 1 kg of main product.
Can the “Ecoinvent LCI calculation tool for crop production” be used for other crops than those that are in the predefined list?
No, it is currently not possible to assess other crops than those that are listed in the data collection sheet. If you need to assess other crops, contact us and we will evaluate the feasibility to add them.
How can I submit data to ecoinvent?
You can import the spold 2 inventory in EcoEditor and submit the data from there to ecoinvent. Please review the following checklist before submitting your data.
If I have comments, suggestions or questions, what should I do?
If you have comments, suggestions or questions, feel free to contact ecoinvent and we will make sure to take them into account and send you an answer.

The Ecoinvent LCI calculation tool for crop production allows you to create in 4 steps a full inventory in Ecospold 2 format. The tool maps the inputs and outputs to ecoinvent exchanges and calculates the field emissions based on models compliant with the ecoinvent database. The import file can be opened and completed in the EcoEditor.

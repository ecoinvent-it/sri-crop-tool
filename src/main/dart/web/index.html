ALCIG - Agricultural Life Cycle Inventory Generator
Easily and quickly generate reliable, precise and specific LCI data for agricultural production
ALCIG is a web-based tool that generates LCI data or crops to be directly imported in LCA software (Simapro). The tool focuses on inventory data for agricultural production. Models used are all publicy available and described in the methodological report.
